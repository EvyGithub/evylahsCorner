# names
names still use -ga/-sa/-se/-szi/-szite, which also means they can take case (name suffix changes za case, not name itself)
change last vowel in suffix if necessary
sub/obj -ø   ga   sa      se      szi
gen     -ow  gow  sajow   sejow   szow
other   -e   ge   saje    seje    sze
basically sa/se add j[case] to not confuse with each other, everything else replaces vowel

example:
[pomeczi] iwano da ben-ge
[pomeczi] iwano amkara ewila-sejow

kjenxajmalsamu


# verbs
ending  description      example
-ucz*   base/inf         ix amu mukucz
-u      everything else  ix amu muku

* -ucz turns verb to noun, so also have to add case

prepositions can be added to verbs to create tenses (verbs by themselves can be any tense, but adding preposition specifies tense)
prep.   tense   example               lankupaje
k       past    ix k muku             I ate
w       during  et w lanu             he/she is eating
t       future  ix te tilu aceja dome³ I will go to your house

³ you don't need t after tilu because it's basically implied what case [aceja dome] will be
  pretty stupid, but so am I

these aren't required. for example if date is specified (check #distelbolszekil), then adding preposition will be pointless

za can be used to mean "for [verb]" or "(in order) to [verb]"
